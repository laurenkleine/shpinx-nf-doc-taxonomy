Software Requirements
=====================

To run this pipeline with Docker the following dependencies are required:

•	Java 7/8

•	`Nextflow 0.24.x <https://www.nextflow.io/>`_ (or higher)

•	`Docker 1.10 <https://www.docker.com/>`_ (or higher)
